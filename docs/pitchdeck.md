# Project Pitchdeck

## Problem
*   Traditional Real World Assets (RWAs) are illiquid and difficult for global investors to access.
*   Integrating fiat (IDR/USD) directly with crypto protocols introduces massive KYC/AML hurdles.

## Solution
*   A decentralized, KYC-free Solana RWA protocol.
*   Uses USDC and IDRX as the base settlement layer, shifting the fiat-to-crypto burden to the users via external off-ramps (P2P/CEX).
*   Enables global access to fractionalized, yield-bearing assets safely and instantly.
*   **Secondary Market Trading:** Supports third-party DEXes (like Raydium and Orca) for immediate liquidity, allowing users to swap their RWA tokens or add liquidity to earn trading fees.
*   **AI Trading Assistant:** An integrated conversational AI that can analyze RWA metrics, answer user questions, and execute investment or swap transactions immediately via intent-based function calling.

## Interoperability & Fiat Access

*   **How users get in:** Fiat-to-crypto conversion is handled at the edge by embedded on-ramp partners — the protocol never touches fiat.
    *   🇮🇩 **IDRX.co** — IDR → IDRX for the Indonesian retail market.
    *   🌍 **Transak / MoonPay** — Card/bank → USDC in-app widget; KYC handled by partner, not the protocol.
    *   🇺🇸 **Coinbase Onramp** — Institutional-grade USD → USDC for US investors.
*   **Cross-chain reach:** Users on Ethereum, Polygon, or any EVM chain can bridge USDC to Solana via **Wormhole** or **DeBridge** in a single click — no need to be a native Solana user.
*   **Net result:** A user in Jakarta, New York, or Berlin can go from fiat to yield-bearing RWA tokens in under 5 minutes, with the protocol remaining fully permissionless and KYC-free.
